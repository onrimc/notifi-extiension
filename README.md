# Notifi 📝

A minimal, persistent note-taking Chrome extension designed for quick access and simplicity.

## 🚀 Features

- 🧠 Quick note taking directly inside the browser
- 💾 Persistent storage using Chrome Storage API
- 🎯 Minimal and distraction-free UI
- 🗑️ Trash system with restore functionality
- 🔍 Search and filter notes
- 📤 Import / Export notes
- ✅ Multi-select and bulk actions

## 🛠️ Installation

### Load as unpacked extension:

1. Open Chrome and go to: chrome://extensions/
2. 2. Enable **Developer Mode**
3. Click **Load unpacked**
4. Select this project folder
